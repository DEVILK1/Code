# 最大公约数：

求证：gcd(a, b) = gcd(b, a % b) 当且仅当 a > b > 0

证明：

    设:a = kc
       b = kd
       c、d互质
       ∴c > d
       ∴gcd(a, b) = k
         a - b = k(c - d)
       ∴ gcd(b, a - b)
         = gcd(kd, k(c - d))
       
       假设 d、(c - d)不互质
       即 d|(c - d)
       ∴设c - d = k'd
       ∴c = d(k' + 1)
       与题设“c、d互质”不符，
       ∴d与(c - d)互质
       
       ∴gcd(kd, k(c - d)) = k
       ∴gcd(a, b) = gcd(b, a - b)
       ∴gcd(a, b) = gcd(b, a % b)
